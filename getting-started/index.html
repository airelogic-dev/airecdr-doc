    Getting Started

    The following page describes the permissions available within the system. Multiple roles can be created which each have different permissions; you should configure your roles in such a way that the permissions match up with tasks that staff at your location are expected to complete.

    Permissions

    System features and their corresponding permissions

    There are many permissions within the system which allow users to access features and specific actions within those features. For example, the 'View Stock Counts' permission will allow a user to view the 'Stock Counts' feature but they will not be able to perform a stock count unless they have the 'Perform Stock Count' permission. See below for descriptions of the individual permissions.

    Add Events

    Allows users access to the 'Returned Drugs' feature at a central hub, and to add stock events throughout the rest of the system.

    Add Returns

    Allows users to add returns.

    Check In Pouches

    Allows users to access the 'Check In' feature of the system and check in pouches.

    Check Out Pouches

    Allows users to access the 'Check Out' feature of the system and check out pouches.

    Collect Stock Returns

    Allows users to collect stock returns from within the 'Drug Returns' feature.

    Create Notes

    Allows users to add notes to stock requests.

    Destroy Dooped Drugs

    Allows users to mark dooped drugs as destroyed.

    Destroy Stock Returns

    Allows users to mark stock returns are destroyed.

    Dispatch Delivery

    Allows users to dispatch a delivery from a location.

    Get Pouches

    Allows users to perform pouch searches and view lists of pouches wherever they appear in the system.

    Manage Drugs

    Allows users to add or edit a drug name, schedule, colour and active status.

    Manage Locations

    Allows users to add or edit a location name, preferred delivery day and default stock levels.

    Manage Notifications

    Allows users to access the 'Notifications' feature and to edit the list of email addresses to which notifications are sent.

    Manage Pouches

    Allows users to access the 'Pouches' feature and to create and update numbered pouches, and manage pouch types.

    Manage Roles

    Allows users to access the 'Roles' feature of the system and to create and edit staff roles.

    Manage Users

    Allows users to access the 'Users' feature of the system, to create and edit users, and to clear a PIN or set a new one.

    Perform Stock Count

    Allows users to perform a stock count from the 'Stock Counts' feature.

    Raise Stock Request

    Allows users to access the 'Stock Requests' feature of the system and to raise a stock request.

    Receive Delivery

    Allows users to access the 'Record Delivery' feature and to receive a delivery.

    Receive Stock Returns

    Allows users at a central hub to receive stock returns.

    Record Ad Hoc Stock Event

    Allows users to record ad-hoc events from the 'Book View' feature.

    Replenish Pouches

    Allows users to replenish pouches within the 'Replenish' feature.

    Replenish Stock

    Allows users to access the 'Replenish' feature.

    Update Event

    Allows users to update an incident number from the 'Book View' feature.

    View Events

    Allows users to access the 'Book View' feature.

    View Stock Counts

    Allows users to access the 'Stock Counts' feature.

    View Stock Requests

    Allows users to access the 'Stock Requests' feature.

    View Stock Returns

    Allows users to access the 'Stock Returns' feature.
