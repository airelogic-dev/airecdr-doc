    Changelog

    Welcome to the Airelogic controlled drugs register changelog. Here you can find out more about recent Aire CDR releases and what the changes mean for you as a Aire CDR user.

    July 7, 2022

    Version 1.1.0 Additional features to ensure legal compliance

    • Allowed global admins to set location on new set up
    • Changed logo
    • Allowed users to select multiple roles
    • Added count of logged in users
    • Improved layout
    • Allowed decimal amounts (instead of whole numbers)
    • Allowed administration of "0" amount of drug
    • Required confirmation if user clicks 'Start Again' after replenishing
    • Added ability to record notes and incident number during stock counts
    • Changed font colour of stock counts in books to red
    • Require stock count of replenished / checked-in drugs
    • Added ability to transfer control of pouches
    • Ensured correct pouch is selected when changing stock if two pouches of the same type are checked out
    • Allowed recording of administrations when not able to replenish pouches
    • Added witnessing of unexpected stock counts during replenishment / check-in
    • Added coloured border around site when not on production
    Sept 23, 2021

    Version 1.0.0 Optimisations and improvements.

    • Allowed leading zeros on PIN entry pad
    • Added expiry time for session when using PIN login
    • Added ability to edit a pouch without changing its name
    • Added prompt for user to sign out after some actions
    • Improved colour contrast and size of text / interactive elements
    • Configured numeric inputs to use a numeric keyboard
    • Improved PIN management user interface
    • Added UI hints
    • Added ability to search users page
    Sept 2, 2021

    Version 0.0.5 Additional features

    • System now automatically adds broken/dooped items to returns on check-in
    • Implemented mutual TLS for PIN sign-in
    • Added tenant-based files for IDP configuration
    • Doop jars can now be recorded as received
    • Added ability to mark return type as destroyed
    • Added ability to delete roles
    • Changed roles UI to dropdowns instead of text entry
    • Fixed note formatting
    • Stock count is now requested when replenishing drugs
    • Added ability to rename an existing role
    • Made sign out button more accessible
    • Added ability to create a user without an invitation
    Jul 26, 2021

    Version 0.0.4 Bug fixes and improvements

    • Fixed issue with case sensitivity for drug IDs
    • Users can record a supplier against a delivery date
    • Allows users to enter name in lowercase for pin sign in
    • Users can edit drug colours for drug book view
    • Roles and permissions are now saved correctly on the user management screen
    • Fixed issues caused by wrong permissions being assigned to certain actions
    • Add signatures on collection of drug returns
    • Improved UI on pouches page
    • Made sign in icons clickable
    • Prevent user from finishing and checking out before completing required steps
    • Broken drugs are returned to stock on check in
    • Improved usability of numeric input boxes
    • New feature to check in and destroy drugs at central hub
    • Corrected stock counts during check in and check out
    • Added progress indicators throughout site
    Jul 05, 2021

    Version 0.0.3 Pouch types, book colours and drug returns

    • Introduced pouch types and a new way for managing pouches
    • Pouches are now tracked across location so OMs can see which pouches have moved
    • Book view now displays drug specific colours to make identification easier
    • Drugs can now be marked for return to the stock hub
    • Weekly delivery schedule information can now be viewed and managed
    • Fridge temps are recorded as part of a stock check
    • PIN based log in is now available
    Jun 11, 2021

    Version 0.0.2 User management and more improvements

    • Implemented a new home page layout
    • Added ability to remove recipients in notifications management
    • Stock discrepancies are now highlighted in the drug book view
    • Added basic user management functionality
    • Other drug schedule now available in drugs management
    • Origin and destination of stock events now shown in book view
    • Add ability to edit location names
    Jun 01, 2021

    Version 0.0.1 General bug fixes and improvements

    • Added paging to the stock requests list to view older requests.
    • Added ability to clear the user filter selection on the drug events page.
    • Fixed some spelling and display problems.
    • Improved system handling of checking out multiple pouches.
    • Removed the ability to dispatch drugs for a stock request from the same location it was created.
    • Improved the UI of warnings in the system.
    • Redesigned the home page layout.
    • Ordered the locations alphabetically in the notifications management page.
    • Added a page to notify the user when there has been an error in the system.
    • Improved the logging of system errors to aid investigation.
    • Improved retry attempts in larger database operations to reduce errors.
